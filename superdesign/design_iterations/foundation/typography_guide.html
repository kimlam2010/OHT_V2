‚ôø Typography Accessibility

WCAG 2.1 AA Compliance
  • All text meets minimum contrast ratios (4.5:1 for normal text, 3:1 for large text)
  • Font sizes are scalable up to 200% without loss of functionality
  • Line heights provide adequate spacing for readability
  • Font families are optimized for screen reading

Best Practices

  • Use Inter for body text and user interface elements
  • Use JetBrains Mono for code, data, and technical specifications
  • Use Space Grotesk for headings and prominent text
  • Maintain consistent font weights within each family
  • Ensure adequate contrast between text and background colors
  • Use appropriate line heights for different text sizes
