Implementation Guide

How to use these templates in your OHT-50 pages:

HTML Structure

<div class="page-container">
  <header class="page-header">
    <!-- Header content -->
  </header>
  
  <main class="page-main">
    <!-- Main content -->
  </main>
  
  <footer class="page-footer">
    <!-- Footer content -->
  </footer>
</div>

CSS Classes

  • .page-container - Main grid container
  • .page-header - Header section
  • .page-main - Main content area
  • .page-footer - Footer section
  • .logo - Logo component
  • .nav-menu - Navigation menu
  • .status-indicators - Status display

Best Practices

  • • Always include skip link for accessibility
  • • Use consistent spacing with CSS variables
  • • Include emergency controls in header when needed
  • • Show system status in footer for monitoring pages
  • • Ensure touch targets are at least 44px on mobile
  • • Test responsive behavior across all breakpoints
