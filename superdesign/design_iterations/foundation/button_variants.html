Button Usage Guidelines

✅ Do's

  • • Use primary buttons for main actions
  • • Use secondary buttons for alternatives
  • • Use safety colors for critical actions
  • • Include icons for better recognition
  • • Provide clear, action-oriented labels
  • • Use appropriate sizes for context

❌ Don'ts

  • • Don't use danger colors for non-critical actions
  • • Don't make buttons too small for touch interfaces
  • • Don't use vague or unclear labels
  • • Don't overload buttons with too much text
  • • Don't use too many button variants on one page
  • • Don't forget to handle loading states
