The OHT-50 spacing system is based on an 8px base unit for consistent, predictable layouts. This system ensures visual harmony and proper content hierarchy in industrial interfaces.

‚úÖ Spacing Best Practices

Consistent Spacing Guidelines

  • Always use design tokens (var(--space-*)) instead of hardcoded values
  • Use smaller spacing (XS, SM) for tight, compact layouts
  • Use medium spacing (MD) for standard component padding
  • Use large spacing (LG, XL) for component separation
  • Use extra large spacing (2XL, 3XL) for major sections
  • Maintain consistent spacing within related components
  • Adapt spacing for different screen sizes using responsive design

Implementation Checklist
